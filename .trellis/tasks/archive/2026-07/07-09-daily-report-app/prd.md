# 日报生成 App (Tauri2)

## Goal

开发一个跨平台桌面应用：用户在左侧输入「今天做了什么」，通过用户自配的 OpenAI 兼容 API 与自定义提示词模板，在右侧**流式**生成符合预期格式的日报 Markdown，支持实时预览、复制、导出 `.md`、历史记录。

## Background

- 目标平台：桌面优先（Windows / macOS / Linux），移动端为后续阶段（架构须兼容）。
- 用户使用 OpenAI 兼容格式的 LLM 服务（OpenAI / DeepSeek / 通义 / Moonshot / 本地 Ollama 等皆可）。
- 日报默认格式为三段式：工作事项 / 遇到问题 / 总结（见样例），但提示词模板可由用户在界面自定义。

日报样例：
```markdown
7.9日报：
### 工作事项
1、重构 dify 放款审核申请编排，简化编排流程
2、审核规则获取新增过滤条件，排除不需要的审核规则
3、开发 AI 审核失败时向发起方发送邮件通知功能
4、联系客户配合 dify 联调 uat 系统测试
### 遇到问题
dify 联调 uat 系统时网络一直不通，联调进度受阻，需进一步排查网络/环境连通性问题。
### 总结
dify 编排重构的核心是做减法——精简流程节点、用过滤条件剔除无关审核规则，既降低维护成本也减少无效执行
```

## Requirements

### 功能需求
- **F1 设置 - API 配置**：BaseURL、API Key（密码框 + 显示切换）、模型名、测试连接按钮。
- **F2 设置 - 提示词模板**：可编辑文本，支持 `{{input}}`、`{{date}}` 变量；一键恢复默认模板。
- **F3 设置 - 导出目录**：可选配置一个目录；留空则每次导出弹窗选位置。
- **F4 主页 - 输入**：左侧文本框输入今日工作要点。
- **F5 主页 - 生成与预览**：调用 LLM 流式生成，右侧 Markdown 实时预览（打字机效果）。
- **F6 主页 - 操作按钮**：生成、重新生成、清空、复制到剪贴板、导出 `.md`。
- **F7 导出**：文件名默认 `yyyy-mm-dd.md`，写入导出目录（已配置）或弹窗选择位置。
- **F8 历史记录**：生成成功后自动保存一条；列表按时间倒序；支持复用（回填输入）、查看、删除。
- **F9 日期变量**：`{{date}}` 默认填今天，正文格式 `月.日`（如 `7.9`）；导出文件名用 `yyyy-mm-dd`。

### 非功能需求
- **N1 跨平台**：Windows / macOS / Linux 桌面。
- **N2 安全**：API Key 存本地、不明文显示；LLM 调用走 Rust 后端，Key 不进入前端 JS 运行时（配置表单除外）。
- **N3 本地持久化**：配置 / 历史本地保存，断网也能打开 app（生成需联网）。
- **N4 错误处理**：API 失败、网络错误、流式中断须有清晰提示，不崩溃。

## Constraints
- 技术栈：Tauri 2 + Svelte 5 + TypeScript + Vite + TailwindCSS。
- LLM 接口：OpenAI 兼容（`POST {baseURL}/chat/completions`，支持 `stream:true`）。
- 单个可编辑提示词模板（不做多模板切换）。
- 仅桌面端 MVP；移动端为后续阶段，但 Rust 后端架构须保证可复用。

## Out of Scope (MVP)
- 移动端打包发布。
- 多模板切换、多 API 配置 profile。
- 深色主题（先浅色，后续可加）。
- 多用户 / 云同步。
- 周报 / 月报等其它类型（模板可手写覆盖，但不专门做）。

## Acceptance Criteria
- [ ] AC1 应用可在 Windows 启动，主界面左输入 / 右预览布局正常。
- [ ] AC2 设置页能填入并保存 BaseURL / Key / 模型 / 导出目录 / 提示词模板，重启后配置仍在。
- [ ] AC3 测试连接按钮对有效配置返回「成功」，对错误配置返回明确错误信息。
- [ ] AC4 填入今日工作要点，点生成，右侧流式逐字显示日报 Markdown，格式符合模板。
- [ ] AC5 生成的日报可一键复制到剪贴板，内容完整正确。
- [ ] AC6 导出生成 `yyyy-mm-dd.md` 文件到配置目录（或弹窗选择），内容与预览一致。
- [ ] AC7 每次生成成功后在历史记录中新增一条；可复用（回填输入）、查看、删除。
- [ ] AC8 API Key 在主界面 / 历史中不明文显示。
- [ ] AC9 网络 / API 错误有友好提示，不崩溃。
- [ ] AC10 提示词模板修改后立即影响下次生成；恢复默认可用。
